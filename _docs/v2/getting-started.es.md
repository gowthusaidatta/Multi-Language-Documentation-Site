# Empezando con la versión 2.0

La versión 2.0 introduce varias mejoras respecto a la versión 1.0, incluyendo mejor rendimiento y nuevas funcionalidades.

## Novedades en la versión 2.0

- **API mejorada** - Endpoints simplificados
- **Mejor rendimiento** - Respuestas 30% más rápidas
- **Documentación mejorada** - Guías y ejemplos más detallados
- **Búsqueda mejorada** - Búsqueda de texto completo en toda la documentación

## Configuración rápida

Para empezar con la versión 2.0:

1. Consulte la [Introducción](/es/docs/v2/introduction)
2. Revise la [Documentación de API](/api-reference)
3. Explore casos de uso de ejemplo

## Características clave

La versión 2.0 se centra en:
- Mejoras en la API
- Optimización del rendimiento
- Mejor experiencia del usuario
- Seguridad mejorada

Para obtener información detallada, consulte la [Introducción](/es/docs/v2/introduction).
