# Bien démarrer avec la version 2.0

La version 2.0 introduit plusieurs améliorations par rapport à la version 1.0, notamment de meilleures performances et de nouvelles fonctionnalités.

## Nouveautés de la version 2.0

- **API améliorée** - Points d'accès simplifiés
- **Meilleures performances** - Temps de réponse 30% plus rapides
- **Documentation améliorée** - Guides et exemples plus détaillés
- **Recherche améliorée** - Recherche textuelle intégrale dans toute la documentation

## Configuration rapide

Pour commencer avec la version 2.0:

1. Consultez l'[Introduction](/fr/docs/v2/introduction)
2. Vérifiez la [Documentation API](/api-reference)
3. Explorez des cas d'utilisation d'exemple

## Caractéristiques principales

La version 2.0 se concentre sur:
- Améliorations de l'API
- Optimisation des performances
- Meilleure expérience utilisateur
- Sécurité améliorée

Pour plus d'informations, veuillez consulter l'[Introduction](/fr/docs/v2/introduction).
