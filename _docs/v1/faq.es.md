# Preguntas Frecuentes

## Preguntas Frecuentes

### Preguntas Generales

**P: ¿Qué es esta plataforma?**
R: Es una API de entrega de contenido de alto rendimiento diseñada para aplicaciones escalables.

**P: ¿Qué lenguajes de programación se admiten?**
R: Admitimos JavaScript/TypeScript, Python, Java, Go y Ruby.

**P: ¿Hay un nivel gratuito?**
R: Sí, ofrecemos un nivel gratuito con 10,000 solicitudes por mes.

**P: ¿Cuál es el SLA?**
R: Garantizamos 99,9% de tiempo de actividad.

### Preguntas Técnicas

**P: ¿Cómo auténtico mis solicitudes?**
R: Usa tu clave API en el encabezado de autorización:
```bash
curl -H "Authorization: Bearer TU_CLAVE_API" https://api.platform.com/...
```

**P: ¿Cuál es el límite de velocidad?**
R: Nivel estándar: 1,000 solicitudes/minuto. Nivel premium: 10,000 solicitudes/minuto.

**P: ¿Se admiten webhooks?**
R: Sí, los webhooks se admiten para notificaciones de eventos en tiempo real.

**P: ¿Cómo se encriptan mis datos?**
R: Todos los datos se encriptan en tránsito con TLS 1.3 y en reposo con AES-256.

### Facturación y Soporte

**P: ¿Puedo cancelar mi suscripción en cualquier momento?**
R: Sí, no hay contratos a largo plazo.

**P: ¿Qué métodos de pago aceptan?**
R: Aceptamos todas las tarjetas de crédito principales, PayPal y transferencias bancarias.
