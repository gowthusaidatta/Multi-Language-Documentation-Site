# Questions Fréquemment Posées

## FAQ

### Questions Générales

**Q: Qu'est-ce que cette plateforme?**
R: C'est une API de distribution de contenu haute performance conçue pour les applications évolutives.

**Q: Quels langages de programmation sont supportés?**
R: Nous supportons JavaScript/TypeScript, Python, Java, Go et Ruby.

**Q: Y a-t-il un niveau gratuit?**
R: Oui, nous offrons un niveau gratuit avec 10 000 requêtes par mois.

**Q: Quel est le SLA?**
R: Nous garantissons 99,9% de disponibilité.

### Questions Techniques

**Q: Comment authentifier mes requêtes?**
R: Utilisez votre clé API dans l'en-tête d'autorisation:
```bash
curl -H "Authorization: Bearer VOTRE_CLÉ_API" https://api.platform.com/...
```

**Q: Quel est le limite de débit?**
R: Niveau standard: 1 000 requêtes/minute. Niveau premium: 10 000 requêtes/minute.

**Q: Les webhooks sont-ils supportés?**
R: Oui, les webhooks sont supportés pour les notifications d'événements en temps réel.

**Q: Comment mes données sont-elles chiffrées?**
R: Toutes les données sont chiffrées en transit avec TLS 1.3 et au repos avec AES-256.

### Facturation et Support

**Q: Puis-je annuler mon abonnement à tout moment?**
R: Oui, il n'y a pas de contrats à long terme.

**Q: Quels sont les modes de paiement acceptés?**
R: Nous acceptons toutes les grandes cartes de crédit, PayPal et les virements bancaires.
