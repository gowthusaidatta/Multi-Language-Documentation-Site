# FAQ

## Frequently Asked Questions

### General Questions

**Q: What is this platform?**
A: This is a high-performance content delivery API that enables developers to build scalable applications with ease.

**Q: What programming languages are supported?**
A: We support JavaScript/TypeScript, Python, Java, Go, and Ruby. SDKs are available for each language.

**Q: Is there a free tier?**
A: Yes, we offer a free tier with 10,000 requests per month. Upgrade anytime as your needs grow.

**Q: What's the SLA?**
A: We guarantee 99.9% uptime with automatic failover and redundancy.

### Technical Questions

**Q: How do I authenticate my requests?**
A: Use your API key in the Authorization header:
```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.platform.com/...
```

**Q: What is the rate limit?**
A: Standard tier: 1,000 requests/minute. Premium tier: 10,000 requests/minute.

**Q: Do you support webhooks?**
A: Yes, webhooks are supported for real-time event notifications. See the API Reference for details.

**Q: How is my data encrypted?**
A: All data is encrypted in transit with TLS 1.3 and at rest with AES-256.

### Billing & Support

**Q: Can I cancel my subscription anytime?**
A: Yes, there are no long-term contracts. Cancel anytime from your dashboard.

**Q: What payment methods do you accept?**
A: We accept all major credit cards, PayPal, and wire transfers.

**Q: How do I get technical support?**
A: Contact us at support@platform.com or use the chat support in your dashboard.

**Q: Are there discounts for enterprise plans?**
A: Yes, contact our sales team at sales@platform.com for enterprise pricing.

### Getting Started

**Q: How long does it take to get started?**
A: You can be up and running in 5 minutes. Just create an account, get your API key, and make your first request.

**Q: Do you have code samples?**
A: Yes, check our [Quick Start](/docs/v1/quick-start) guide and API Reference for examples.

**Q: Can I test before committing?**
A: Absolutely! Use our free tier or sandbox environment to test before upgrading.
